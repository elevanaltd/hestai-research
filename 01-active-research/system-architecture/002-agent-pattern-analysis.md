# Agent Cognitive Pattern Analysis Report

Analysis of 54 agent files in `/Users/shaunbuswell/.claude/agents/` reveals distinctive structural patterns and cognitive frameworks that enable consistent agent activation. This research identifies how RAPH (Read-Absorb-Perceive-Harmonise) directives create cognitive crystallization, the relationship between constitutional foundations and operational behavior, and key patterns that distinguish successful from problematic agent implementations.
